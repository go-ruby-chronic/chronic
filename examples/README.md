# Ruby examples

Pure-Ruby examples for `chronic` (natural-language date/time parsing), provided
by [go-embedded-ruby](https://github.com/go-embedded-ruby/ruby) (rbgo). Run them
with the `rbgo` interpreter:

```sh
rbgo examples/chronic_usage.rb
```

| File | Shows |
| --- | --- |
| [`chronic_usage.rb`](chronic_usage.rb) | `Chronic.parse` of relative/absolute phrases against a fixed "now". |

Each example is executed as-is under rbgo (`require "chronic"`).
